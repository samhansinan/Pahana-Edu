# Pahana-Edu System

## Features
- Login authentication
- Add/Edit/Delete Customer Accounts
- Manage Items
- Generate Bills
- Help Section

## Technologies Used
- Java Servlets & JSP
- MySQL
- JDBC
- HTML/CSS
- 3-Tier Architecture
- DAO & DTO Design Patterns

## How to Run
1. Clone the repository
2. Set up MySQL DB with `pahanaedu` schema
3. Configure DB credentials in `DBConnection.java`
4. Deploy to Tomcat
